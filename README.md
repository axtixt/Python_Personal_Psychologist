Project Overview

  SoulChat Psychologist is an AI-powered interactive psychological analysis application that combines natural language processing, speech synthesis, and social media data analysis to provide users with personalized mental state assessments and life advice.

Core Features

  1. Multimodal Input Analysis
  Text Input: Users can directly input text to describe their current mental state or issues.
  
  Social Media Integration: Optional connection to X (formerly Twitter) and Instagram accounts to analyze users' posting history.
  
  2. Intelligent Psychological Analysis
  NLP Deep Analysis: Utilizes advanced natural language processing techniques to understand user input.
  
  Emotional State Assessment: Identifies the user's current emotional state (levels of anxiety, depression, stress, etc.).
  
  3. Personalized Responses
  Real-Time Interactive Dialogue: Provides instant psychological support based on the SoulChat analysis engine.
  
  Voice Output: Converts analysis results and recommendations into natural speech feedback.
  
  4. Smart Report Generation
  Comprehensive Analysis Report: Generates personalized reports based on user input and social media data.
  
  Recommendation Plan: Offers practical and actionable suggestions for psychological adjustment and lifestyle improvement.

Technical Architecture
  Frontend Interface
  User input panel (text/voice)
  
  Social media account connection interface
  
  Real-time conversation display
  
  Backend Services
  NLP Processing Module: Sentiment analysis, intent recognition, semantic understanding.
  
  Voice Processing Module: Speech synthesis (TTS).
  
  Data Analysis Module: Social media data crawling and analysis.
  
  Report Generation Engine: Personalized recommendations and report generation.

Integrated Technologies
  Large Language Model (LLM): Powers dialogue and psychological analysis.
  
  Sentiment Analysis API: Professional emotional state assessment.
  
  Social Media APIs: Data retrieval from X and Instagram.
  
  Speech Synthesis Engine: High-quality voice output.

Installation and Deployment
  Environment Requirements
  Python 3.8+
